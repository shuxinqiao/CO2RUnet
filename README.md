# RU-net PyTorch Reproduction

Train and Test dataset download:

https://drive.google.com/drive/folders/1nHpXhKjOw_nBk8GvoYfzBj5tcBNaZ6Hh


## Usage

1. Get dataset
2. Convert dataset by `hdf5_dim_convert.ipynb` If dataset not transformed as `(N,C,H,W)`
3. Use `train.ipynb` for training and sample test


## Environment
Python Version: 3.10.12 (main, Nov 20 2023, 15:14:05) [GCC 11.4.0]

CUDA Availability: False

PyTorch Version: 2.1.2+cpu
